# svg-kit

A lightweight SVG icon kit packaged for easy reuse in JavaScript / TypeScript projects.

This package exposes a small, curated set of SVG assets that can be imported directly and used in your apps without additional tooling or icon libraries.

## Installation

#### GitHub Package
```bash
npm install @soumil4561/svg-kit
```
Make sure your project is configured to use GitHub Packages:

#### .npmrc
```bash
@soumil4561:registry=https://npm.pkg.github.com
```

## Usage
You can import the built files directly:

#### JavaScript / TypeScript
```js
import icon from '@soumil4561/svg-kit';
```
